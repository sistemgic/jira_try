# 🤖 AI Learning README

Welcome! This is a random but practical guide to help you start learning Artificial Intelligence (AI). Whether you're a beginner or someone refreshing your knowledge, this README gives you direction, resources, and a simple roadmap.

---

## 📌 What is AI?

Artificial Intelligence is the field of building systems that can perform tasks that normally require human intelligence, such as:

* Understanding language
* Recognizing images
* Making decisions
* Learning from data

---

## 🧠 Core Areas of AI

### 1. Machine Learning (ML)

* Supervised Learning
* Unsupervised Learning
* Reinforcement Learning

### 2. Deep Learning (DL)

* Neural Networks
* CNNs (Computer Vision)
* RNNs / Transformers (NLP)

### 3. Natural Language Processing (NLP)

* Text classification
* Chatbots
* Language models (like GPT)

### 4. Computer Vision

* Image classification
* Object detection
* Image generation

## 🧪 Example Code (Python)

```python
from sklearn.linear_model import LinearRegression

model = LinearRegression()
X = [[1], [2], [3]]
y = [2, 4, 6]

model.fit(X, y)
print(model.predict([[4]]))
```
